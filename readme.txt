Easily add PDF Invoices and Packing Slips to your WooCommerce Store

== Description ==
If you are looking for a WooCommerce Invoice & Packing Slip plugin then look no further! A super easy to use plugin that will provide your WooCommerce store with great looking PDF Invoices & Packing slips

Set up is super-easy and you will quickly have Invoices & packing slips added to your WooCommerce store.

= Key Features =
* Automatically add Invoices to any or all of the standard WooCommerce emails - they get automatically attached to your customer emails
* Use built in order numbering or create your own custom Invoice number - super easy!
* Easily select the paper size that works for you (A4 or Letter)
* Customize Invoice template
* Manually generate invoices on demand
* View an invoice right from within the Order

This a very easy to use plugin and setup.

You will have Invoices added to your store in minutes

== Installation ==


1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're good to go to start using the plugin
